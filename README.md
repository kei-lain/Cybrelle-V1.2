# Cybrelle-web

## About Cybrelle
Cybrelle is a cloud based vulnerability scanner and it allows for the user to add up to 15 hosts to analyze vulnerabilities that are present on each system. From network and system vulnerabilities and the web application spits out CVEs pertaining to the programs and systemd units loaded.  Additionally, with the use of AI, it will give the user information regarding how to fix each one. 

A commandline version will come soon. 
## Features
OpenAI's GPT4 API is being used to aid in giving recommendations on how to alleviate vulnerabilities in the configuration files on each device.
